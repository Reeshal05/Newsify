Newsify — Personalized News Recommendation System

Newsify is a web application that provides users with news articles based on their interests. The aim of the project is to make news consumption more relevant by showing content that matches user preferences.

About the Project:
This project was developed as a final year group project as part of our undergraduate program.
This repository is a fork of the original project created by my teammate. I used this version to work on and understand different parts of the system.

Tech Stack
Frontend: React
Backend: Node.js
Database: MongoDB
API: NewsAPI
Features
Personalized news feed based on user interests
Onboarding to collect initial user preferences
Real-time news using NewsAPI
Basic scoring system based on relevance and recency
How It Works
Users select their interests during signup
News articles are fetched using NewsAPI
Articles are ranked using a scoring mechanism
Recommendations update based on user interaction
